## v2.0 可视化-图形

![](Assets/神经网络可视化示图.png)

> 使用"view图形化"来显示整个网络的缩影;
> 1. 可操作
> 2. 动态扩展

* 说明
  - NVDelegate_He是He的可视化代理实现范例,同样你也可以写你自己的代理实现,来支持你们自己的神经网络;

* 项目结构为:
  * 网络的总可视化窗口:NVView
  * 一个网络含多个模块:NVModuleView
  * 一个NVModuleView含多个节点:NVNodeView
  * 所有节点间,可能包含多个关联NVLine
